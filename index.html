Bem-vindo ao Sistema de Solicitações

O FormSubimit foi desenvolvido com o objetivo de facilitar a gestão de solicitações escolares no SENAI. A plataforma oferece uma maneira prática e ágil para realizar solicitações, como mudanças de horários, desligamentos de alunos e outros processos administrativos.

O que é o FormSubimit?

O FormSubimit é uma plataforma online simples e eficiente, projetada para facilitar a comunicação entre funcionários e a administração do SENAI. Através de um sistema intuitivo, ele permite que você faça solicitações de maneira rápida, sem burocracia, garantindo um atendimento mais eficiente e sem complicações.

Com o FormSubimit, você pode acessar o sistema de forma simples, preencher os formulários necessários e enviar as suas solicitações para os responsáveis. O sistema oferece um ambiente amigável para que você gerencie suas solicitações de forma autônoma, rápida e segura.

Como utilizar o FormSubimit?

Siga estas etapas simples para realizar sua solicitação:

  1. Escolha o tipo de solicitação: No menu lateral, selecione a opção de solicitação que deseja fazer.
  2. Preencha o formulário: Complete o formulário com as informações solicitadas. Garanta que todos os dados sejam inseridos corretamente.
  3. Visualize o formulário: Após preencher o formulário, clique em "Visualizar" para gerar uma versão preenchida do formulário padrão.
  4. Verifique as informações: Revise todas as informações inseridas no formulário para garantir que estão corretas.
  5. Gere o PDF: Após verificar, clique no botão "Salvar em PDF" para gerar o formulário finalizado em formato PDF.
